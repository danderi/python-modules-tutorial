# 🐍 Aprende Tutorial de Módulos de Python

¡Bienvenido al Tutorial de Módulos de Python! 🎉

En este tutorial, aprenderás todo lo que necesitas saber sobre los módulos de Python, desde importaciones básicas hasta la creación de tus propios módulos. ¡Este tutorial es perfecto para principiantes que quieren organizar mejor su código y usar las increíbles funciones integradas de Python!

### Lo que aprenderás

- Qué son los módulos y por qué son súper útiles
- Cómo importar y usar módulos integrados de Python
- Crear tus propios módulos personalizados
- Mejores prácticas para trabajar con módulos
- Ejemplos del mundo real y ejercicios prácticos

### Estructura del Tutorial

Este tutorial está dividido en temas principales (como "01-intro-a-los-modulos") y ejercicios prácticos más pequeños (como "01.1-tu-primer-import"). Cada sección se basa en lo que aprendiste antes, ¡así que es mejor seguirlas en orden!

## Abrir en Codespaces

Puedes abrir directamente este tutorial en [Codespaces](https://codespaces.new/?repo=4GeeksAcademy/python-modules-tutorial) para un entorno fácil basado en la nube. Esto te permite comenzar a trabajar en los ejercicios sin necesidad de configurar nada en tu máquina local.

## Instalación Local

1. Asegúrate de tener [LearnPack](https://learnpack.co) instalado, node.js versión 14+, y Python versión 3.7+, NPM (6 o superior). Este es el comando para instalar LearnPack:

```bash
$ npm install @learnpack/learnpack -g
```

2. Clona este repositorio:

```bash
$ git clone https://github.com/4GeeksAcademy/python-modules-tutorial
$ cd python-modules-tutorial
```

3. Inicia el tutorial:

```bash
$ learnpack start
```
