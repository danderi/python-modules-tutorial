# 🎯 Your First Import

Let's start with something simple! We will import the `random` module and use it to generate a random number.

## 📝 Instructions

1. In app.py, import the native Python module `random`
2. Use `random.randint(1, 10)` to generate a random number between 1 and 10 and store the returned result in a variable `random_number`
3. Print the random number

## ✅ Expected Output

You should see a random number between 1 and 10 printed on the console!
```python
# Example output (your number might be different):
7
```

## 💡 Hint

- The import statement looks like this:

```python
import random
```

- Remember that after importing, you need to use the module name before its functions, like `random.randint()`.
