# 🎯 Your First Import

Let's start with something simple! We're going to import the `random` module and use it to generate a random number.

## 📝 Instructions

1. In app.py, import the `random` module
2. Use `random.randint(1, 10)` to generate a random number between 1 and 10
3. Print the random number

## 💡 Hint

The import statement looks like this:
```python
import random
```

## 🔎 Important

Remember that after importing, you need to use the module name before its functions, like `random.randint()`.

You should see a random number between 1 and 10 printed to the console!

# Example output (your number might be different):
