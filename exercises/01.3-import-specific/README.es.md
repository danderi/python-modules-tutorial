# 🎯 Importando partes específicas

A veces no necesitas todo de un módulo, ¡solo quieres una o dos cosas! Aprendamos a importar elementos específicos. 

## 📝 Instrucciones

1. Del módulo `random`, importa solo la función `choice`
2. Usa la función `choice()` para seleccionar y mostrar aleatoriamente una fruta
3. Imprime el resultado para mostrar la fruta seleccionada aleatoriamente.

## 💡 Pista

- Para importar solo un elemento de un módulo, usa:

  ```python
  from random import choice 
   ```

- Para usar la funcion `choice()` lo hariamos de la siguiente manera:

  ```python
    selected_fruit = choice(fruits)
  ```
