# 🎯 Importando partes específicas

A veces no necesitas todo de un módulo, ¡solo quieres una o dos cosas! Aprendamos a importar elementos específicos. 

## 📝 Instrucciones

1. Del módulo `random`, importa solo la función `choice`
2. Crea una lista de frutas: "apple", "banana", "orange", "grape"
3. Usa la función `choice()` para seleccionar y mostrar aleatoriamente una fruta

## 💡 Pista

Para importar solo un elemento de un módulo, usa:

  ```python
  from random import choice 
   ```
