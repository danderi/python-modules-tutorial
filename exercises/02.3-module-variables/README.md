# 🌟 Module Variables and Constants

Modules can have variables too! Let's learn how to create and use them.

## 📝 Instructions

1. Create a module called `constants.py`
2. Add some variables:
   - `PI = 3.14159`
   - `GRAVITY = 9.81`
   - `AUTHOR = "Your Name"`
3. Import and use these constants in your code
