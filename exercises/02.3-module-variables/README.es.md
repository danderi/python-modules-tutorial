# 🌟 Variables y constantes del módulo

¡Los módulos también pueden tener variables! Aprendamos cómo crearlas y usarlas.

## 📝 Instrucciones

1. En tu `app.py`, define estas constantes:
    - `PI = 3.14159`
    - `GRAVITY = 9.81`
    - `AUTOR = "Tu Nombre"`
2. Crea funciones que usen estas constantes
3. Importa y usa estas constantes en tu código

## 💡 Pista

Las constantes generalmente se escriben en letras MAYÚSCULAS:
```python
PI = 3.14159
GRAVITY = 9.81
```
