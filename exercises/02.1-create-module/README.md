# 🎨 Create Your First Module

Let's create a simple greeting module! We'll make a module that has functions to say hello in different ways.

## 📝 Instructions

1. Create a file called `greetings.py`
2. In app.py, write a function called `say_hello(name)` that returns "Hello, [name]!"
3. Write another function called `say_goodbye(name)` that returns "Goodbye, [name]!"
4. Import and use these functions in another file
