# 🏗️ Creando tus propios módulos

¡Hola! 👋 Ahora que sabes cómo usar módulos existentes, ¡aprendamos a crear los nuestros!

## 🤔 ¿Por qué crear módulos personalizados?

1. 📦 Agrupar código relacionado
2. 🔄 Compartir código entre diferentes archivos
3. 🎨 Mantener tus proyectos organizados

## 💡 Cómo funciona

Crear un módulo es súper sencillo:
1. Crea un nuevo archivo Python (que termine en .py)
2. Escribe tus funciones y variables en él
3. ¡Impórtalo en otros archivos al igual que lo hiciste con los módulos integrados!

## 🌟 Mejores prácticas

- Dale a tu módulo un nombre claro y descriptivo
- Mantén las funciones relacionadas juntas
- Agrega comentarios para explicar qué hace tu código
- Usa buenos nombres de variables y funciones
